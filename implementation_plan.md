# Crypto Signals Web App - Implementation Plan

## Goal
Create a crypto signals web app that provides buy/sell recommendations based on various indicators. The initial phase focuses on creating the UI and fetching real-time BTC price.

## Tech Stack
- **Framework**: Vite + React
- **Styling**: Vanilla CSS (Premium aesthetics: Dark mode, Glassmorphism, Animations)
- **Data Source**: Public Crypto APIs (e.g., CoinGecko, Binance) for real-time price.

## Phase 1: Setup & Basic UI (Current Focus)
1.  [x] Initialize Vite project.
2.  [ ] Clean up default boilerplate.
3.  [ ] Create a premium layout structure (Header, Main Dashboard, Footer).
4.  [ ] Implement `PriceFetcher` component to get real-time BTC price.
5.  [ ] Style the application with a modern, dark-themed design.

## Phase 2: Signals & Analysis (Future)

- Implement basic chart pattern recognition (or mock it for now).
